
# Gestión de Datos 2do Cuatrimestre UNLaM

Este proyecto consiste en una recopilación de ejercicios correspondientes a la cursada del 2do cuatrimestre 2025 de Gestión de Datos (Bases de Datos II), llevada a cabo como parte del plan de estudios de la Tecnicatura en Desarrollo de Aplicaciones Móviles en la Universidad Nacional de La Matanza.

## Instalación

Clonar el repositorio:
```bash
git clone https://github.com/Tomaselbert2000/Gestion-de-Datos---Ejercicios-con-Microsoft-SQL-Server.git
```


### Pre-requisitos
Windows: recomendado [SQL Server Management Studio](https://learn.microsoft.com/es-mx/ssms/install/install)

Linux: [Visual Studio Code](https://code.visualstudio.com/) en conjunto con las [extensiones](https://marketplace.visualstudio.com/items?itemName=ms-mssql.mssql) oficiales de Microsoft para SQL (reemplazan a SMSS en Linux)

Base de datos de muestra utilizada: [Adventure Works 2014](https://github.com/microsoft/sql-server-samples/releases/download/adventureworks/AdventureWorksDW2014.bak)
## Ejercitación
Los archivos presentan una serie de ejercicios ordenados por dificultad incremental, en los cuales se abordan primeramente temas como el uso de funciones del sistema, JOINs, HAVING, GROUP BY, etc. Por otra parte, más adelante se muestran también herramientas como procedimientos almacenados, triggers y cursores.

## Extras
Como agregado a los ejercicios, vas a encontrar una serie de apuntes y resúmenes teóricos referentes al uso del lenguaje SQL, focalizado en Microsoft SQL Server y contando también con una sección dedicada a la seguridad en el manejo de bases de datos.
